## 介绍

id-pointer-sdk 是基于Java语言开发的项目，是工业互联网标识解析体系的客户端软件开发工具包，主要提供对工业互联网标识服务的标识解析，标识管理等功能。

## 状态

首个版本开发阶段

## 下载

通过Maven下载最新的jar

```xml
<dependency>
  <groupId>cn.teleinfo.id-pointer</groupId>
  <artifactId>id-pointer-sdk</artifactId>
  <version>2.2.1</version>
</dependency>
```
